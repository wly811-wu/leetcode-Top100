# Deque

//双端队列，两端都可以进出，当做栈用
//是queue的子接口

Deque<Integer>  deque = new Deque<>()；
dequeue.offerFirst(1);//队头入列
dequeue.offerlast(2);//队尾入列
dequeue.pollFirst();//队头出队，（并移除）
dequeue.pollLast();//队尾出队（并移除）
dequeue.peekFirst();//查看队头
dequeue.peekLast()//查看队尾
