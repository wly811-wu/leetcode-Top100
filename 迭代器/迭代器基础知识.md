# 遍历List并删除指定元素，用迭代器实现，不能用普通for循环

/**
     * 删除列表中所有等于 target 的元素
     * @param list 原始列表（会被原地修改）
     * @param target 要删除的目标元素
     * @return 删除后的列表（也可以定义为 void）
     */
    public List<String> removeTarget(List<String> list, String target) {
        // 使用迭代器安全遍历并删除
        Iterator<String> iterator = list.iterator();
        while (iterator.hasNext()) {
            String current = iterator.next();
            // 使用 equals 比较（注意 null 安全，这里假设 target 不为 null）
            if (current.equals(target)) {
                iterator.remove(); // 迭代器的 remove 方法
            }
        }
        return list;
    }
